# Flower Shop Management System

## Description
This project is a Flower Shop Management System using MySQL and Python (Tkinter GUI).

## Features
- Manage flower inventory
- Manage customers
- Track orders and payments
- Billing system

## Technologies Used
- Python (Tkinter)
- MySQL

## How to Run

1. Install Python
2. Install MySQL
3. Run SQL file:
   flower_shop.sql

4. Install required package:
   pip install mysql-connector-python

5. Run Python file:
   python flower_shop.py

## Note
Update MySQL password in the Python file before running.
